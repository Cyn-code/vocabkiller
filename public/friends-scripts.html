Workflow

Episode reader first, role-play typing second.

Every episode opens into a dedicated page where you can scan the full script, filter dialogue by character, and then launch role-play typing from the currently selected dialogue only.

1. Open an episode

Each imported transcript lives as local JSON under the site, so browsing does not depend on GitHub at runtime.

2. Filter by character

Character buttons are derived from the episode dialogue. Multi-speaker lines only match if every speaker is selected.

3. Start role-play typing

The episode page converts the current filtered dialogue into a sentence-practice style typing flow with previous and next line context.
